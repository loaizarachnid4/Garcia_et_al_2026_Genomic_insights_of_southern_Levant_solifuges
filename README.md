# Description

**Tree_Files**: 

All individual tree files generated and summarized in Figure 2. 

**SNP_alignments**: 

SNP alignments for each of the six focal solifuge families generated in the study. We utilized the _Gluvia dorsalis_ genome (SRA: ERS16539865; Lozano-Fernandez et al., 2024) to pull SNPs for Solpugidae & Daesiidae (Suborder: Australosolifugae) and we used the _Paragaleodes pallidus_ genome published in this study for Gylippidae, Karschiidae, Rhagodidae, and Galeodidae (Suborder: Boreosolifugae). The alignments and structure-formatted files here were generated using BWA (Li H. and Durbin R. 2009)).

**R_code**: 

Includes modified code from Pyron (2023) for single-layer sSOMs for analyzing SNP data for small sample sizes. For original code visit: https://github.com/rpyron/delim-SOM

**COI_alignments**: 

Contains unaligned and aligned mitochondrial COI sequences derived from our sequence data using Mitofinder (Allio et al. 2020)
